CSS

  1. 웹 페이지에 CSS를 삽입하는 2가지 방법
    1)Style 태그: 웹브라우저에게 style태그 안쪽에 있는 내용은 CSS문법에 맞게 처리하도록 지시하는 HTML 태그임.
    처리할 대상(모든 a 태그)과 처리효과(글씨 색상을 red)를 기술해 줌.
    ex) [style 태그]
    a { color: red; }
    [/style]
    2)Style 속성: HTML의 Style 속성을 쓰면 웹브라우저는 CSS의 문법에 따라 해석해서
    Style속성이 위치하고 있는 태그에게만 그 속성의 값을 적용하개 해줌.
    Ex) <태그 style="color:red;">
  2. text-decoration:값이 none인 경우 a태그인 경우 참조 표시인 밑줄이 없어지고, underline인 경우 밑줄이 그어짐
  3. .class속성 값: .saw{효과}라고 하면 class속성값이 saw인 모든 태그에 대해 효과를 나타내라는 것
    같은 class속성값이 즉, .saw{효과}가 여러개 있는 경우 코딩된 순서대로 효과가 나타남
  4. #id속성 값: #active{효과}라고 하면 id속성값이 active인 모든 태그에 대해 효과를 나타내라는 것
    여기서 #id속성은 .태그 속성보다 우선순위가 높으므로 .class가 #id보다 나중에 코딩되었어도 #id 속성이 효과를
    나타내게 됨. 참고로 id속성의 값(active)은 웹페이지 내에서 1번만 써야함.그래서 우선순위가 제일 높음
    우선순위: id선택자 > class 선택자> 태그 선택자
    공통적으로 디자인 할것은 태그선택자로 하고 예외를 두어야 하는 것들은 class와 id선택자로 지정함
    w3s 사이트에서 "css selector"로 검색해보면 여러가지 선택자가 나옴.
