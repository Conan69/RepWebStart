태그

1. p태그: 문단을 이루도록 시각화 해주기 위해 p태그 전후로 1라인씩 띄워줌.
긴 내용을 br태그로 라인을 띄우는 것 보다는 문단이라는 정보를 코드상에 나타내줌.

2. H1~H6태그: "코딩"이란 단어에 이 태그를 사용해 줄 경우 사람들이 브라우저에서 "코딩"이란 단어로 웹검색을 하는경우, 웹브라우저는 "코딩"이란 단어에 Head 태그를 사용한 웹페이지를 상위에 보여줌.
단순히 "코딩"이란 단어를 진하게 하고 폰트크기를 크게하는 경우에도 웹페이지 상에서는 동일하게 보이나 웹검색시 브라우저에서는 내 웹 페이지를 찾아주지 않음.

3. Li태그: 리스트를 작성할때 사용하는 태그로 어디서 부터 어디까지가 해당 리스트 인지를 나타내주는
부모태그(ul / ol)와 같이 사용됨 ol은 자동으로 리스트 번호를 주므로 이 태그를 사용하면 됨.

4. Title태그: 웹 브라우저 탭에 나타나는 내용으로 웹 페이지 제작시 응축된 내용으로 정해야
타인 검색시 상단에 올라올수 있음.

5. 한글이 깨져 나올때: "meta charset=utf-8" 사용.
웹브라우저에게 이 웹페이지를 열때 utf-8로 열라는 소리임

6. "!DOCTYPE html" : 이 문서는 HTML 이라고 관용적으로 써줌.
"HTML태그": 전체를 묶는 최 고위층 태그.
"Head태그": 본문에 대한 설명을 해주는 부분
"Body태그": 본문을 묶음

7. "a href="index.html">[Home] 태그" : a(anchor)태그는 링크를 걸어주는 태그로
"웹주소" or "현 폴더내의 html 파일명"을 적어주면 됨.
